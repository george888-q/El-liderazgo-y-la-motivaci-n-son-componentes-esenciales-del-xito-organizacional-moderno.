# Blog de Liderazgo y Motivación

Blog simple con animaciones sobre liderazgo y motivación transformando la motivación del grupo en productividad.

## Características

- Diseño responsive y moderno
- Animaciones al hacer scroll
- Videos embebidos de YouTube
- Efectos hover interactivos
- Navegación suave entre secciones

## Archivos

- `index.html` - Estructura principal del blog
- `styles.css` - Estilos y animaciones
- `script.js` - Funcionalidades JavaScript

## Uso

Simplemente abre `index.html` en tu navegador para ver el blog.
